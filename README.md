💰 Expense Tracker (Python)

📌 Project Description
The Expense Tracker is a simple Python application that helps users record their expenses and calculate the total amount spent. It is a beginner-friendly project that demonstrates the use of lists, loops, user input, and built-in Python functions.
✨ Features
Add multiple expenses.
Store expenses in a list.
Calculate the total expense automatically.
Display all entered expenses.
Easy to understand and use.
🛠️ Technologies Used
Python 3
📂 How It Works
The user enters the number of expenses.
The program asks for each expense amount.
All expenses are stored in a list.
The program calculates the total using the sum() function.
The expense list and total amount are displayed.
